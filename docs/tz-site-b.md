# Техническое задание — Сайт Б (Сайт-заявка / лидогенерация)

## 1. Назначение

Сайт Б — сайт-заявка / лидогенерация.

**Назначение:** пользователь оставляет заявку на подбор ноуската, задает вопрос или получает консультацию.

**Тематика:** подбор ноускатов из Китая, покупка под конкретный автомобиль, консультации по совместимости, доставка и поставка, помощь в подборе деталей.

Нужны типовые шаблоны страниц, чтобы сайт работал как понятная воронка:
**вход → доверие → выбор → заявка → подтверждение**

## 4. SEO и навигация

- На страницах списков и хабов — **нумерованная пагинация** (не бесконечная подгрузка)
- Формат: `1 2 ... 5 6 7 ... 10` с переходом по номерам
- Кнопка "Показать ещё" допустима только как дополнительный вариант
- Страницы переключаются блоками для индексации Яндексом

---

## 5. Footer и юридические элементы

Footer обязателен на **всех страницах** и должен содержать:

- Политику конфиденциальности
- Согласие на обработку персональных данных
- Cookie-политику
- Настройки cookie
- Служебные и контактные ссылки

### Cookie-баннер

- Обязателен при первом заходе на сайт
- Пользователь может: принять cookie, отклонить необязательные, открыть настройки
- В настройках можно отключать аналитические cookie
- Яндекс.Метрика загружается **только после согласия** пользователя

## 6. Адаптив

- Desktop макет: **1200 px** (основной)
- Tablet: **768 px**
- Mobile: **360 px**
- Desktop 1440 px — не обязателен

### Требования к адаптиву

- Без горизонтального скролла
- Карточки и блоки перестраиваются
- Mobile — полноценный, не урезанный
- На внутренних страницах — breadcrumbs
